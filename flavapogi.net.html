Project Cost Report: flavapogi.net

Overview

This report outlines the detailed cost breakdown for developing flavapogi.net, a web and mobile app system for managing mess services for students, admins, and mentors. The system includes features like QR code integration, Firebase-based authentication, payment gateway, and food management. The development is led by Amresh Bhuyan, a full-stack developer.

1. Development Charges

Component Charge (₹) Type Description
Domain & Hosting Setup 5000 Per Year The cost includes setting up a domain (₹1239) and hosting on Firebase Blaze Plan (∼₹3761/year for scaling).
Authentication System 8000 One-Time Implementation of Firebase Authentication for secure user login using email and Google accounts.
Web App Development 30,000 One-Time Creating a responsive web app for managing food items, menus, QR codes, and payments.
Mobile App Development 35,000 One-Time Developing mobile apps for iOS and Android using React Native for on-the-go access to the system.
iOS App Store Hosting 10,000 Per Year Annual fee for an Apple Developer Program subscription to host the iOS app on the App Store.
Backend Development 25,000 One-Time Setting up Firebase services (Firestore, Realtime Database, Cloud Functions) for data storage and logic.
Payment Gateway Integration 12,000 One-Time Adding and configuring a secure payment gateway like Razorpay or Stripe for handling transactions.
QR Code Integration 10,000 One-Time Implementing QR code functionality for food scanning and payment verification.
UI/UX Design 15,000 One-Time Designing an intuitive and professional user interface for both web and mobile platforms.
Testing & Deployment 10,000 One-Time Testing the system for bugs and errors, followed by deployment to Firebase Hosting and app stores.
Additional Maintenance 14000 3 Months Maintenance post-launch for updates, bug fixes, and support.
Total Development Charges ₹1,74,000
